# Package index

## Core workflows

Discover, download, and open BlueTopo assets.

- [`bluertopo()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo.md)
  : Download and extract NOAA BlueTopo bathymetry
- [`bluertopo_download()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_download.md)
  : Download original NOAA BlueTopo assets for an AOI
- [`bluertopo_tile_polygons()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_tile_polygons.md)
  : Retrieve all BlueTopo tile polygons
- [`bluertopo_tiles()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_tiles.md)
  : Discover BlueTopo tiles intersecting an AOI

## Policies and cache

Configure native-resolution selection and package cache behavior.

- [`bluertopo_resolution()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_resolution.md)
  : Construct a BlueTopo native-resolution policy
- [`bluertopo_cache_dir()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_cache_dir.md)
  : Locate the bluertopo cache directory
- [`bluertopo_cache_clear()`](https://el-cordero.github.io/bluer-topo/reference/bluertopo_cache_clear.md)
  : Clear package-owned cache content
